# Kintai

```sh
Usage: kintai --file-path <FILE_PATH> <COMMAND>

Commands:
  start
  stop
  summary
  help     Print this message or the help of the given subcommand(s)

Options:
  -f, --file-path <FILE_PATH>
  -h, --help                   Print help
  -V, --version                Print version

```

{
  "records": {
